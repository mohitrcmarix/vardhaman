== Description ==

CuraMedix is a clean, responsive WordPress theme for clinics, hospitals and medical professionals to showcase services, staff and portfolios. Fully compatible with Elementor and includes a demo homepage built with Elementor. Features: customizable colors, responsive layouts, and translation readiness.
